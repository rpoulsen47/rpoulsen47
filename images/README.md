# Images folder

Drop your headshot here as `headshot.jpg` (or whatever name you set in CONFIG).

Project images go in `projects/`. Suggested naming: short-project-name.jpg.

## Image guidelines

- **Headshot:** square crop, around 800×800 px
- **Project images:** landscape, around 1200×900 px (4:3 ratio looks best with the layout)
- Keep files under 300 KB for fast loading. Use https://squoosh.app to compress.
